# Food Delivery Platform Analysis

## Project Overview

This project analyzes a **food delivery platform dataset** to understand customer ordering behavior, restaurant performance, and delivery trends. The analysis integrates multiple relational datasets to uncover patterns in user activity, order distribution, and restaurant popularity.

The goal of this project is to demonstrate **data analysis, SQL-based data integration, and visualization techniques** to extract meaningful business insights from real-world datasets.

---

## Dataset

The project uses multiple datasets representing different components of a food delivery platform:

* **users.csv** – customer information and user activity
* **orders.csv** – order records and timestamps
* **order_details.csv** – item-level details for each order
* **restaurants.csv** – restaurant metadata
* **menu.csv** – menu items and pricing
* **delivery_partner.csv** – delivery partner information
* **food.csv** – food category data

These datasets were combined to create a **relational analytical dataset**.

---

## Objectives

* Analyze **customer ordering behavior**
* Identify **popular restaurants and menu items**
* Understand **order distribution and platform activity**
* Explore **relationships between users, orders, and restaurants**
* Generate **data-driven insights through visualization**

---

## Key Analysis Performed

### Data Cleaning

* Processed and cleaned datasets using **Pandas**
* Handled missing values and inconsistent records
* Prepared structured datasets for analysis

### Data Integration

* Combined multiple datasets using **SQL joins**
* Built relational links between **users, orders, restaurants, and menu items**

### Exploratory Data Analysis (EDA)

* Analyzed **order patterns and user behavior**
* Identified **top restaurants and frequently ordered items**
